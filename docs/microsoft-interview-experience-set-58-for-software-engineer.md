软件工程师海得拉巴微软面试。

第一轮:(1 小时)
1。给定 3 个排序数组。在 O(N)时间内找到所有数组的公共元素，并且没有额外的空间。如果没有，返回 false。
2。双链表:1 到下一个节点，1 到随机节点。复制这个列表。可以使用额外的空间。

第二轮:(45 分钟)
1。一个数(x**n)在 O(log N)时间内的幂
2。给定一个除了两个元素之外都有重复的数组。找到他们。我们总是有 2 个这样的元素。
